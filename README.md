# Simulador de PDV em PHP CLI

Aplicação de terminal desenvolvida para simular o fluxo de um caixa de mercado. O foco do projeto é a aplicação de conceitos sólidos de PHP moderno.

## Funcionalidades
* **Entrada de Dados:** Cadastro interativo de produtos via `readline`.
* **Validação:** Verificação de tipos numéricos para evitar erros de cálculo.
* **POO:** Uso de Classes e `ArrayObject` para manipulação eficiente da lista de compras.
* **Formatação:** Geração de cupom fiscal visualmente alinhado utilizando manipulação de strings (`str_pad`).

## Tecnologias
* PHP 8+ (`declare(strict_types=1)`)
